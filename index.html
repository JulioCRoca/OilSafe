Herramienta frontend para estimar la viscosidad dinámica de mezclas de aceite SAE 20W-50 a partir de pérdidas de carga en un sistema de tuberías en serie. El cálculo se ejecuta en el navegador con Pyodide, sin servidor externo.

Mezclas de trabajo

Aceite puro, aceite + gasolina y aceite + gasolina + carbonilla.

El selector asigna automáticamente densidad y gravedad específica para usar esos datos en el balance de energía.

Modelo base
hL,obj = ΔPρ·g + Δz + hA μ = ρ · ν

1. Datos globales del sistema

Define la mezcla, el sistema de unidades, la energía disponible y la cantidad de tramos. El balance se plantea entre tanques; como sus áreas son muy grandes, se asume v1 ≈ v2 ≈ 0, por eso no se pide velocidad en los tanques. Para tanques abiertos a la atmósfera, usa ΔP = 0. Si no hay bomba, usa hA = 0.

A Propiedades del fluido

La densidad y la gravedad específica se asignan de forma invisible al cálculo.

B Balance de energía

Supuesto del balance entre tanques: A1 y A2 son mucho mayores que el área de la tubería, por tanto v1 ≈ v2 ≈ 0. Las velocidades que aparecen en la iteración son internas de los tramos y solo se usan para calcular pérdidas por fricción y accesorios.

2. Tramos de tubería en serie

Cada bloque representa un tramo. El caudal es el mismo en todos los tramos; el cálculo desacopla el factor de fricción por tramo y ajusta la viscosidad cinemática hasta igualar la pérdida de carga objetivo.

C Datos dinámicos por tubería

3. Resultados

La tabla muestra todas las iteraciones realizadas. El valor final de viscosidad dinámica se calcula como μ = ρ·ν.

Método de iteración usado

Estimación inversa de viscosidad mediante método de iteración de punto fijo multivariable con desacoplamiento por bloques y relajación. Para cada ν asumida se calculan Rn,i, fi con Swamee-Jain, pérdidas mayores, pérdidas menores y hL,calc; luego se ajusta ν hasta igualar hL,obj.

Fórmula del factor de fricción aplicada

Para el régimen turbulento se usa la ecuación explícita de Swamee-Jain, con rugosidad ε, diámetro Di y número de Reynolds Rn,i:

fi = 0.25 [ log10 ( ε3.7Di + 5.74Rn,i0.9 ) ]2

Notas de uso

Convención de ΔP: se usa directamente la fórmula hL,obj = ΔPρ·g + Δz + hA. Ingresa ΔP con el signo que corresponda al planteamiento entre punto inicial y punto final.
Unidades internas: aunque selecciones sistema inglés, el cálculo convierte todo a SI y reporta μ en Pa·s, que es la unidad principal para viscosidad dinámica.
